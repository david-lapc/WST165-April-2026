CSS Position Property Interactive Demo

This interactive demo helps you understand how the CSS position property affects element layout. Select different position values from the dropdown to see how the green child element behaves within the gray parent container.

  • static: Default positioning, follows normal document flow.
  • relative: Positioned relative to its normal position, can be offset with top/right/bottom/left.
  • absolute: Positioned relative to the nearest positioned ancestor, removed from normal flow.
  • fixed: Positioned relative to the viewport, stays in place during scrolling.
  • sticky: Hybrid of relative and fixed, sticks when scrolling reaches a threshold.
